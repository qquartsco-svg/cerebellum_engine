# Release Notes - Cerebellum Engine v0.6.0-beta

**릴리즈 일시**: 2026-01-22

---

## 🎉 주요 기능

### 핵심 기능
- **Predictive Feedforward**: 다음 순간의 오차 예측
- **Trial-to-Trial 보정**: 반복 궤적의 미세 편차 제거
- **Variance 감소**: 미세한 떨림 필터링
- **기억 기반 적응**: 해마의 기억을 즉각 행동으로 변환

### v0.6.0-beta 신규 기능
- **Confidence 기반 gain 조절**: 기억의 신뢰도에 따른 적응형 보정
- **Error norm 기반 saturation**: 과도한 보정 신호 방지
- **Context 가중치**: 상황에 따른 적응형 반응
